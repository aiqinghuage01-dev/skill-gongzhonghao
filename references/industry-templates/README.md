# 行业范文库

> 这里放 3-4 个典型行业的完整公众号范文。
> 作用：给 AI 提供"像这样写"的锚点，避免不同行业的学员文章被 AI 写成同一个腔调。

---

## 使用规则（写给 AI）

**Step 1：识别老板行业分类**

读完 `~/Desktop/我的内容库（勿动）/01-黄金五环/` 后，把老板归入以下类别之一：

| 类别 | 匹配行业 | 参考范文 |
|------|---------|---------|
| **本地服务 B2C** | 汽修、美容、家政、本地餐饮、健身房、家装 | `01-本地服务.md` |
| **教培 B2C** | 钢琴、英语、留学、艺术、文化课、考证 | `02-教培.md` |
| **制造 B2B** | 门窗、瓷砖、家具、食品加工、设备 | `03-制造业.md` |
| **知识 IP / 咨询** | 讲师、咨询、教练、财税、法律 | `04-知识付费.md` |

行业归类完成后，读对应范文。**只读一篇**，不要串读。

**Step 2：范文不是模板，是锚**

范文的作用：
- ✅ 学习**结构节奏**（开场怎么钩、中段怎么推、结尾怎么落）
- ✅ 学习**场景具体化**（怎么用细节建画面）
- ✅ 学习**业务植入位置**（第几段自然桥接）

范文**不是**：
- ❌ 拿来改几个词就发的模板
- ❌ 内容的抄写对象

**真实内容**永远从老板的 `01-黄金五环` + `03-人设故事` 来。

---

## 范文编写规格

每篇范文必须满足：
- **字数 1800-2000**
- **六原则 6/6 通过**
- **六维评分 ≥ 105/120**
- **包含完整 HTML 版**（V3 Clean 模板）
- **文末标注"哪些字段来自黄金五环 / 哪些来自人设故事"**，方便 AI 学习"素材→成品"的映射

## 做范文的方法

1. 挑一个**真实学员**（不要虚构），读他的 01-黄金五环 和 03-人设故事
2. 选一个常见选题（行业痛点 / 客户常见误区 / 一个具体案例）
3. 按 7 步骨架写 1800-2000 字
4. 跑三层自检（六原则 / 六维评分 / 一票否决）
5. 标注"哪段用了什么素材"
6. 把文章存到这里

做完一篇 → 用这篇跑测试：让 AI 写同行业另一个学员的文章，看输出质量。
